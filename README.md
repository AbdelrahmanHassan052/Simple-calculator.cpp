Simple CLI Calculator built in C++
This project is a basic calculator that runs in the terminal.
It supports the following operations:

Addition

Subtraction

Multiplication

Division

Power (a^b)


The program uses functions to handle each operation, and a loop that allows the user to perform multiple calculations until they choose to exit.

This project was created to practice:

C++ functions

Loops

Basic input/output

Conditional statements

Simple program structure


It’s a beginner-friendly project and will be improved in the future with more features and better input validation
